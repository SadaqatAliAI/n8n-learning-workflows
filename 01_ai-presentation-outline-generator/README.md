# AI Presentation Outline Generator

## Overview

This workflow generates an AI-powered presentation outline from a user-provided topic.

## Features

- Form submission
- AI-generated presentation points
- Email delivery
- HTML formatting
- Fully automated

## Technologies

- n8n
- OpenAI
- Gmail
- AI Prompt Engineering

## Workflow

1. User fills the form.
2. AI generates presentation points.
3. Email is sent automatically.


## Installation

1. Download workflow.json.
2. Import into n8n.
3. Configure your credentials.
4. Run the workflow.
